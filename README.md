# Guide d'installation
1. Prérequis : Windows 10, 16GB ram, application [Unity Hub](https://login.unity.com/fr/sign-up) instalée, version de l’éditeur « 6000.3.6f1 » sur Unity Hub.
2. Pour pouvoir utiliser ce jeu, vous devez télécharger la [dernière release](https://github.com/HOPPhugo/P_Remed_Appro/archive/refs/tags/V3.0.zip).
3. Puis, décompressez le dossier téléchargé.
5. Créez un projet avec cette version.
6. Une fois le projet créé, remplacez les dossiers « Assets », « Packages » et « ProjectSettings » par ceux que vous venez de télécharger depuis la release.
7. Ouvrez le jeu depuis Unity Hub et jouez-y.
