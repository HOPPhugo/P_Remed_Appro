# Guide d'installation
1. Pour pouvoir utiliser ce jeu, vous devez télécharger la dernière release.
2. Puis, décompressez le dossier téléchargé.
3. Allez sur Unity Hub et vérifiez que vous avez la version de l’éditeur « 6000.3.6f1 ». Si ce n’est pas le cas, téléchargez-la.
4. Créez un projet avec cette version.
5. Une fois le projet créé, remplacez les dossiers « Assets », « Packages » et « ProjectSettings » par ceux que vous venez de télécharger depuis la release.
6. Ouvrez le jeu depuis Unity Hub et jouez-y.
